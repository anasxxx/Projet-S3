📌 ParaEnsias – Application de gestion parascolaire
ParaEnsias est une application web innovante conçue pour améliorer la communication entre les clubs étudiants et l’ADEI. Ce projet vise à créer un outil de liaison digital dédié à la gestion des activités parascolaires, en répondant aux défis actuels de communication.

🔧 Fonctionnalités principales :

Gestion des annonces

Suivi et gestion des dépenses des clubs

Système de réclamations pour les étudiants

Interface utilisateur fluide et intuitive

💡 Technologies utilisées :

Java EE

CSS

Javascript

🎯 Ce projet met l’accent sur l’efficacité, la fluidité de l’expérience utilisateur et l’optimisation des ressources. Il offre aux étudiants un canal direct pour s’exprimer et améliorer la vie associative au sein de l’ENSIAS.
